# BrainStack - Flashcard Learning App

A modern, object-oriented flashcard learning application built with Python. - Uses AI to give users currated practice tests based on your flashcards.





## Getting Started

1. Run `python main.py`
2. Open `http://localhost:5000` in your browser
3. Create your first deck
4. Add some flashcards
5. Start studying!

The app will automatically create the data directory and initialize with default settings.
